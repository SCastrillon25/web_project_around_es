# Tripleten web_project_around_es

## Proyecto del Bootcamp de TripleTen

Proyecto enfocado en el posicionamiento de elementos y diseño responsivo, desarrollado utilizando HTML y CSS. Se aplicaron técnicas de maquetación como Flexbox y Grid para lograr una correcta distribución de los componentes en distintos tamaños de pantalla, garantizando una experiencia de usuario adaptable y consistente.
